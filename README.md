#  Processo de Isenção de Valores

Esse processo se trata de dois formulários, um para isenção de valores de alunos e candidatos e o outro se trata de uma solicitação de bolsa e benefícios. 
Tendo como obejtivo facilitar o trabalho do setor de contas a pagar.



## Visão Geral
Este processo é destinado a candidatos/alunos que buscam a isenção de taxas e/ou solicitação de bolsa e benefício passando por um fluxo específico da Clariens.
